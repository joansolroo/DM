                                            Guest • 2 years ago

                                            i use this code to calculate future position of target, this code just only for linearly path, when target change direction, bullet miss

                                            Vector totarget = target.position - tower.position;

                                            float a = Vector.Dot(target.velocity, target.velocity) - (bullet.velocity * bullet.velocity);
                                            float b = 2 * Vector.Dot(target.velocity, totarget);
                                            float c = Vector.Dot(totarget, totarget);

                                            float p = -b / (2 * a);
                                            float q = (float)Math.Sqrt((b * b) - 4 * a * c) / (2 * a);

                                            float t1 = p - q;
                                            float t2 = p + q;
                                            float t;

                                            if (t1 > t2 && t2 > 0)
                                            {
                                            t = t2;
                                            }
                                            else
                                            {
                                            t = t1;
                                            }

                                            Vector aimSpot = target.position + target.velocity * t;

                                                          dovyski Guest • 2 years ago

                                                          It's pretty hard to predict things using steering behaviors, because all forces might change along the way. That's what makes steering behaviors so powerful.

                                                          You can estimate a character's future position based on it's current velocity vector though. Take a look at these tutorials for more info on position estimation:

                                                          - http://gamedev.tutsplus.com/tu... (check the pursuit behavior)

                                                          - http://gamedev.tutsplus.com/tu... (check the "ahead" point)
